iLink

Cette application est conçu pour gérer les stations de recharge dans les communautés urbaines. Il détermine les villes de la communauté urbaine qui devraient accueillir des stations de recharge, tout en respectant des contraintes spécifiques telles que l'accessibilité et la minimisation des coûts.

Structure du Projet :

Packages :

- backend : Contient des classes qui gèrent la logique interne des villes et des communautés urbaines.
- exceptions : Contient des exceptions personnalisées utilisées dans le projet.
- frontend : Gère les interactions utilisateur et les interfaces pour manipuler la communauté urbaine.

Utilisation :

Pour exécuter le programme, utilisez la classe "App". La méthode main se trouve dans cette classe. Si un chemin vers un fichier est fourni, le programme utilisera la version en ligne de commande. Sinon, il lancera l'interface graphique.

Étapes :

1. Lancez l'application.
2. Chargez la communauté urbaine avec les villes et leurs noms.
3. Utilisez les menus ou l'interface graphique pour ajouter des routes entre les villes, gérer les stations de recharge et explorer les options.
4. Vérifiez et affichez les villes avec des stations de recharge, en veillant à respecter les contraintes d'accessibilité.

Prérequis :

Pour utiliser l'interface graphique, assurez-vous d'avoir installé les frameworks suivants :
- JavaFX pour les fonctionnalités graphiques. Téléchargez-le à partir de : https://openjfx.io/
- JUNG pour la modélisation et l'analyse basées sur les graphes. Téléchargez-le à partir de : http://jung.sourceforge.net/

Ces frameworks sont nécessaires pour le bon fonctionnement de l'interface graphique.
